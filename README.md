# StackSee Documentation

Official documentation for the StackSee API - Analyze any website's technology stack with a single API call.

## About StackSee

StackSee is a powerful REST API that analyzes websites to extract comprehensive technology stack information, including frameworks, libraries, hosting providers, and branding details.

## Documentation Structure

- **Getting Started** - Introduction and quickstart guide
- **API Reference** - Complete API documentation with endpoints and examples

## Development

This documentation is built with Mintlify. To preview changes locally:

Install the [Mintlify CLI](https://www.npmjs.com/package/mint):

```bash
npm i -g mint
```

Run the development server:

```bash
mint dev
```

View your local preview at `http://localhost:3000`.

## Publishing Changes

Changes pushed to the main branch are automatically deployed to production.
